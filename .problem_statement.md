Problem statement: Project Management Collaboration Tool

- Problem: Build a real-time project management collaboration tool. Users should be able to create projects, assign tasks, send direct messages within the project, create group discussions, and receive notifications for task updates and messages.
- Technologies: TypeScript, Angular Framework, C#, Web API, ASP.NET Core
- Requirements:
    - Real-time updates on project status and task changes
    - User authentication and project access management
    - Task assignment and progress tracking
    - Group discussion functionality with message threads
    - Notification system for task updates and new messages
